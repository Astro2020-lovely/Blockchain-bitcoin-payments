A) Config File - top

This script works with PHP 5 or more. The first thing to do is: Edit the blockchain_config.php file e.g 

					
//RECEIVE PAY BLOCKCHAIN DETAILS
//Your Own APIKey received from Blockchain
$app_api_key = "your-api-key-goes-here";
//(Where you will receive payment) Your Xpub key in Blockchain  (located in Settings -> Addresses -> Manage -> More Options -> Show xPub)
$app_xpub = "your-xpub-key-goes-here";
//this can be anything you want as has to be unique to you 
//password used to request has to be the one for confirmation
$app_secret = "your-secret-key-goes-here"; //e.g secret123456
//http://yourdomain.com"; //example https://yourdomain.com  or in directory then http://yourdomain.com/directoryname
$app_rootURL = "http://yourdomain.com/"; //e.g add something http://yourdomainname.com/
//$orderID - This can be anything from Database User_ID or Invoice - or simply a Unique ID being paid for (OrderID,UserId,UpgradeId)
$orderID = uniqid();//will be based on your liking like $invoice_id from Database
//Amount in BTC - amount can be from a database invoice/plans
$amount = "0.00077657";
